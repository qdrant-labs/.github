# Qdrant Labs

This is the home for experimental projects, community-driven tools, and works-in-progress built around [Qdrant](https://github.com/qdrant/qdrant) — the high-performance, composable vector search engine built in Rust.

You'll find libraries, integrations, and early-stage tools that extend the Qdrant ecosystem. Some are actively developed. Others are proofs of concept.

> ⚠️ **Disclaimer:** Repositories in this organization are experimental and community-driven. They are **not officially supported** by Qdrant and may be incomplete, unmaintained, or subject to breaking changes without notice. For production use, refer to the [official Qdrant repositories](https://github.com/qdrant).
